# Understanding Diffusion Models

A code-first, visual walkthrough of diffusion models built on top of:

> *"Understanding Diffusion Models: A Unified Perspective"* Calvin Luo, Google Research (2022)

The goal is to make every concept in the paper interactive and reproducible math explained in plain language, every idea backed by runnable code.
